# Calliop v0.1.0

Première release publique de Calliop — dictée vocale locale pour Windows (prioritaire).

## Fonctionnalités

- Dictée globale (Alt + Espace) dans toute application Windows
- Transcription locale Whisper (small / distil-fr-dec16)
- Auto-edits IA optionnels via Qwen3 local (sidecar llama.cpp)
- Overlay, tray, démarrage automatique
- Dictionnaire personnel, snippets, historique FTS
- Contexte par application (ton adapté)
- Onboarding et paramètres complets
- Insight : latence, mots/min, usage par app

## Installation (Windows)

1. Téléchargez `Calliop_*_x64-setup.exe` (NSIS) ou le `.msi`.
2. Installez WebView2 si nécessaire.
3. Acceptez l’avertissement SmartScreen (binaire non signé en v1).
4. Au premier lancement, le modèle Whisper se télécharge (~466 Mo).

Voir [docs/installation.md](../docs/installation.md).

## macOS / Linux (expérimental)

Les artefacts DMG, AppImage et deb sont fournis **sans signature ni tests v1**. Utilisation à vos risques.

## Limitations connues

- Binaires Windows **non signés** (SmartScreen)
- Modèles ML téléchargés séparément au premier usage
- Mises à jour automatiques : opt-in dans Paramètres → Avancé
- Pas de support officiel macOS/Linux en v1
