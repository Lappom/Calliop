# Calliop v0.1.0

First public release of Calliop — local voice dictation for Windows (primary platform).

## Features

- Global dictation (Alt + Space) in any Windows application
- Local Whisper transcription (`small` / `distil-fr-dec16`)
- Optional AI auto-edits via local Qwen3 (llama.cpp sidecar)
- Overlay, tray, auto-start
- Personal dictionary, snippets, FTS history
- Per-app context (adapted tone)
- Onboarding and full settings
- Insights: latency, words/min, usage by app

## Installation (Windows)

1. Download `Calliop_*_x64-setup.exe` (NSIS) or `.msi`.
2. Install WebView2 if needed.
3. Accept the SmartScreen warning (unsigned binary in v1).
4. On first launch, the Whisper model downloads (~466 MB).

See [docs/installation.md](../docs/installation.md).

## macOS / Linux (experimental)

DMG, AppImage, and deb artifacts are provided **unsigned and untested for v1**. Use at your own risk.

## Benchmarks

STT results: [docs/benchmarks.md](../docs/benchmarks.md).

## Known limitations

- Windows binaries are **unsigned** (SmartScreen)
- ML models download separately on first use
- Automatic updates: opt-in in Settings → Advanced
- No official macOS/Linux support in v1
