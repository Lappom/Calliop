# Calliop v0.1.2

Feature and patch release — auto-update fixes, onboarding practice flow, microphone settings, and improved overlay waveform.

## Changes

### Auto-updates

- Fix updater public key alignment between `tauri.conf.json` and the CI signing key
- Enable automatic updates by default (still disableable in Settings → Advanced)
- Restart the app automatically after a successful signed update
- Show a desktop notification when an update download or install fails
- Add `scripts/verify-updater-pubkey.mjs`; CI and release workflows fail fast on pubkey mismatch

### Onboarding

- Replace the dictation “test” step with a guided **practice** flow (EN/FR)
- New practice field, hotkey coach, step indicator, and coach feedback animations
- Clearer instructions and success/error feedback during first-run setup

### Audio & overlay

- Richer waveform visualization using frequency-band data from the capture pipeline
- New waveform physics layer for smoother, more responsive overlay feedback
- Remove obsolete `WaveformStub` component

### Settings

- Choose the **microphone input device** in Settings (with live level probe)
- Localized labels for audio input (English and French)

### Other

- Refactor model download toasts for clearer ordering and modularity
- Improve error handling in the Windows LLM sidecar preparation script

## Important for v0.1.1 users

Installations from **v0.1.1** were built with a mismatched updater public key and cannot auto-update. Please **reinstall manually once** from this release; later updates will then work automatically.

## Installation (Windows)

1. Download `Calliop_*_x64-setup.exe` (NSIS) or `.msi`.
2. Install WebView2 if needed.
3. Accept the SmartScreen warning (unsigned binary in v1).
4. On first launch, the Whisper model downloads (~466 MB).

See [docs/installation.md](../docs/installation.md).

## macOS / Linux (experimental)

DMG, AppImage, and deb artifacts are provided **unsigned and untested for v1**. Use at your own risk.

## Known limitations

- Windows binaries are **unsigned** (SmartScreen)
- ML models download separately on first use
- Automatic updates are enabled by default; disable in Settings → Advanced if needed
- No official macOS/Linux support in v1
